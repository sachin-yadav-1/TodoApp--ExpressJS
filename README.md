### Description
- Express app for a simple todo app.
- API Docs: [Postman Link](https://documenter.getpostman.com/view/13622241/2s83tCKYBq) 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
